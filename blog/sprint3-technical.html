Explain control flow and loops

Loops

Loops are used to run a block of codes until a set of conditions are met. When you want the code to run over and over again with a different value then loop is used. There are different types of loops ‘while loop’, ‘for loop’, ‘do… while’.

We see loop in our everyday life. Similarly, in my life there are loops that I do everyday such as after waking up and brushing my teeth I prepare kids lunch boxes and this loop continues until its week-end.

Control flow

‘If…. else’ statements are referred as control flow. These statements help a program meet the conditions to execute code. If one statement is true then then one block of code is executed. If the statement is false then another block of code is executed.

One of the basic examples is: if a student clears the medical preliminary examination then he/she will get the admission in the medical field else they will try in other departments

Describe what the DOM is

DOM is a Document Object Model. It is a tree model of various elements of HTML. In other words, DOM is the nice representation of the entire web page including margin, padding, and content.

DOM is a document which gives access to the webpage of any website and one can manipulate it temporarily.

When DOM is accessed then we can add, delete or create code.

Explain what functions are and why they are useful.

Functions are backbone of JavaScript. They are a set of statements that performs a task/calculate value. In JS functions are defined by using ‘function’ command.

Functions are used to reuse a code. The programmer once defines the code and then that code can be used again with different arguments.

Functions are used to avoid writing same code again and again.
